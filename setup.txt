This will just be a list/documentation of the libraries we will use so we can keep track of them.
Frontend
    - npm
        - https://www.npmjs.com/get-npm
    - yarn
        - Once you have npm installed, run "npm install --global yarn"
    - react-router-dom
        - once npm and yarn are installed, navigate to the frontend folder
        - run "yarn add react-router-dom"
    - material-ui
        - run "yarn add @material-ui/core"
            - this will only be for the core usage of material ui (buttons, cards, grid etc)
        - to install other packages like icons use:
            - "yarn add @material-ui/icons
        - "yarn add @material-ui/lab"
        - "yarn add material-ui-dropzone"
        - "npm i @date-io/date-fns"
        - "yarn add @material-ui/pickers"
        - "npm install react-chatbot-kit"
 
Backend
    - psycopg2
        - https://pypi.org/project/psycopg2/
        - can also be installed with homebrew
